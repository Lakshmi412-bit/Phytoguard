# Tomato Leaf Disease Detection System

## Project Overview
This project is a web-based application that identifies possible tomato plant diseases from a leaf image.  
The system displays disease information such as symptoms, effects, solutions, prevention methods, and recommended fertilizers.

Currently, the system demonstrates disease identification using predefined disease data and simulated detection logic.

## Technologies Used
React.js
TypeScript (TSX)
React Router DOM
Node.js
Hono Framework

## Dataset Reference

The disease categories and information used in this project are based on the **PlantVillage Tomato Leaf Dataset** available on Kaggle.

Dataset Link:  
https://www.kaggle.com/code/mohammedhamzamoawad/tomato-leaf-disease-detection/input


The dataset contains thousands of labeled images of tomato leaves belonging to different disease classes such as:

- Bacterial Spot
- Early Blight
- Late Blight
- Leaf Mold
- Septoria Leaf Spot
- Target Spot
- Yellow Leaf Curl Virus
- Tomato Mosaic Virus
- Healthy Leaf

These disease classes were used as reference categories for the system.
